# ECOCO 商城權限管理矩陣（RBAC）

Version：1.0

---

# 1. 文件目的

建立商城後台權限管理機制，確保：

* 權責分離
* 個資保護
* 財務資料控管
* 系統安全

---

# 2. 角色定義

| 角色代碼  | 角色名稱  |
| ----- | ----- |
| ADMIN | 系統管理員 |
| MKT   | 行銷人員  |
| CS    | 客服人員  |
| FIN   | 財務人員  |
| WH    | 倉儲人員  |
| AUDIT | 稽核人員  |

# 4. 特殊權限

## 財務

可執行：

* 發票作廢
* 折讓單建立
* 財務報表下載

---

## 系統管理員

可執行：

* 使用者建立
* 權限管理
* API管理
